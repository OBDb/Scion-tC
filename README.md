# Scion tC

This repository contains signal set configurations for the Scion tC, organized by model year and version. The files are structured to allow for easy differentiation between model generations and other vehicle parameters, ensuring accurate signal mapping for each version of the Scion tC.

The Scion tC was a compact sport coupe manufactured by Toyota under its Scion brand from 2004 to 2016 over two generations. Named "touring coupe," the tC was introduced first in the United States for the 2005 model year and then, beginning with the second generation in 2011, in Canada as well. The tC was Scion's best-selling model, constituting almost 40% of total Scion sales.
